# TheCompressor (iOS)

TheCompressor is an iOS PDF compression app.

## Unsigned IPA (for sideload workflows)

Releases may include an **unsigned** IPA named `TheCompressor-unsigned.ipa`.

### Important
- This IPA is **not installable directly** by iOS.
- You must sign it with a sideloading tool/profile before installing.
- Use only your own Apple ID / developer certificates and trusted sideload apps.

## How to sideload (general)
1. Download `TheCompressor-unsigned.ipa` from the latest GitHub Release.
2. Open your sideload app (for example, AltStore, Sideloadly, or similar tools).
3. Import/select the IPA.
4. Let the sideload app sign the IPA with your Apple account/certificate.
5. Install to device.
6. On iOS, trust the developer profile if prompted.

## Security note
Only install builds from trusted sources and verify release assets before installing.
